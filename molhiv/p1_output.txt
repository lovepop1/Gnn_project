WHY BOTH ATOM FEATURES AND BOND STRUCTURE ARE NECESSARY:
  Atom features (pharmacophores): identify specific functional groups such as
  charged nitrogens required for chemical binding affinity. Without knowing
  atom TYPE, the model cannot identify whether a binding site is present.
  Bond structure (geometric fit): a linear chain vs a ring structure interacts
  differently with the HIV enzyme pocket, even with identical atoms. Bond type
  (single/double/aromatic) changes electron density and 3D geometry entirely.
  A model using atoms only cannot reason about molecular shape; a model using
  topology only cannot identify which atoms are pharmacologically active.
  Both conditions must be simultaneously met for a molecule to inhibit HIV.
